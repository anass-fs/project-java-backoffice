# Tableau de bord TechStore

Ce projet est le frontal d'un tableau de bord d'administration pour un site e-commerce. Il permet de gérer les produits, les commandes, les clients, et bien plus encore.

## Fonctionnalités

*   **Tableau de bord**: Vue d'ensemble de l'activité du magasin.
*   **Gestion des produits**: Ajouter, visualiser, modifier et supprimer des produits.
*   **Gestion des commandes**: Suivre et gérer les commandes des clients.
*   **Gestion des catégories**: Organiser les produits par catégories.
*   **Gestion des clients**: Afficher et gérer les informations des clients.
*   **Gestion des utilisateurs**: Gérer les utilisateurs du tableau de bord.
*   **Gestion des factures**: Générer et suivre les factures.

## Technologies utilisées

*   HTML
*   CSS
*   JavaScript

## Lancement

Pour lancer l'application, ouvrez le fichier `index.html` dans votre navigateur web.
